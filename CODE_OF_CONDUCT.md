# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

**Summary**: Be respectful, constructive, and welcoming. Focus on the work. Harassment and exclusionary behaviour are not acceptable.

**Enforcement**: Project maintainer — open a GitHub issue marked `[conduct]` or contact via GitHub if the issue is sensitive.
